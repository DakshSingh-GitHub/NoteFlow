# NoteFlow — AI-Generated Note Taking Web App

> ⚠️ **This project was generated primarily by AI as an benchmark experiment.**

NoteFlow is a modern note-taking web application created as an experiment to test the limits of current AI coding models. The goal of this project is not to present handcrafted production software, but to evaluate how far AI can go in generating a polished, functional frontend application from prompting alone.

This repository serves as a technical exploration of AI-assisted development and modern UI scaffolding.

---

## Overview

NoteFlow features a sleek dark-mode interface and a modular note management system. The application demonstrates AI’s ability to generate:

* Clean component structure
* Modern UI/UX patterns
* Consistent styling and layout
* Functional state-driven interfaces

This project acts as a benchmark and learning artifact rather than a traditional manually engineered application.

---

## Features

* Create, edit, and organize notes
* Category-based organization
* Color tagging system
* Grid and list viewing modes
* Customizable UI settings
* Smooth animations and modern dark theme
* Responsive and minimal interface
