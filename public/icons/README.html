NoteFlow Icon Setup

To generate PWA icons:

  1. Go to favicon.io or PWA Builder Image Generator
  2. Upload the icon.svg file from public/icons/
  3. Download the generated icons
  4. Place them in the public/icons/ folder with these names:
    • icon-72x72.png
    • icon-96x96.png
    • icon-128x128.png
    • icon-144x144.png
    • icon-152x152.png
    • icon-192x192.png
    • icon-384x384.png
    • icon-512x512.png

Or use ImageMagick (if installed):

cd public/icons
for size in 72 96 128 144 152 192 384 512; do
  convert icon.svg -resize ${size}x${size} icon-${size}x${size}.png
done
